#多源海量动态农业数据加工融合工具

本系统基于农业领域构建海量知识的结构及分布模型，对于web信息爬取数据进行结构化、信息化处理，提高了农业领域模型的共享性、可理解性和标准化程度，可以针对部分不确定、不完备、不协调知识的推理，解决农业数据共享和融合等问题。系统主要提供以下功能：

+ 数据加工：农产品价格信息加工、农产品市场信息加工、农产品供求信息加工、农产品平均价格信息加工。
+ 数据融合：农产品价格信息融合、农产品市场信息融合、农产品供求信息融合。
+ 精数据管理：农产品名信息管理、农产品价格信息管理、农产品市场信息管理。
+ 基础数据管理：行政区划管理、农产品类别管理、采集URL管理。


同时，系统提供数据加工、融合的相关工具，包括：数据格式加工、日期格式加工、地域信息加工、计量单位加工、文本相似度计算以及基于投票机制的数据融合工具。


本工具旨在为处理多源海量农业数据，提供农业数据清洗、融合、管理功能。
运行__KP-1.0.jar__即可开启主程序。
